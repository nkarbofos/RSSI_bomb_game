# Игра "Обезвредь бомбы"

### Игровая механика

#### Создание игры

- Игрок А (ищущий) создает лобби и получает код приглашения
- Игрок Б (прячущийся) подключается по коду
- После подключения обоих игроков начинается игра

#### Фаза расстановки (60 секунд)

- Игрок Б размещает на поле 3 передатчика
- Передатчики можно ставить только в свободные клетки
- Передатчики не могут занимать одну клетку
- Таймер 60 секунд отображается на экране
- Если время вышло или отправлена команда досрочного завершения, передатчики при необходимости расставляются случайно

#### Фаза поиска (до 20 ходов)

- Игрок А видит поле 10x10 (или 15x15)
- У игрока А есть 3 приемника (R1, R2, R3)
- Каждый ход игрок А может сделать одно действие:
  - Переместить приемник в любую свободную клетку
  - Добавить новый приемник (если их меньше 3)
  - Убрать приемник (освободить клетку)
  - Запустить одну итерацию градиентного спуска (вычислить позиции передатчиков)
- После каждого действия сервер возвращает:
  - Текущие RSSI значения для всех пар (приемник --> передатчик)
  - Вычисленные приблизительные позиции передатчиков

#### Завершение игры

- Игрок А может завершить игру досрочно (кнопка "Готово")
- Игра автоматически завершается после 20 ходов
- После завершения показывается:
  - Реальные позиции передатчиков
  - Итоговые выбранные позиции
  - Средняя ошибка локализации в клетках

### Технические особенности

#### Сервер

- Написано на C++17/20, Boost.Asio
- Асинхронная обработка подключений
- Поддержка до 100 одновременных игр
- Таймеры для фазы расстановки (60 сек)
- Хранение состояния игр в памяти
- Логирование всех действий для отладки

#### Клиент

- Qt 6 (Quick)
- Отображение игрового поля (QML Grid)
- Отдельный поток для сетевых запросов (QThread + Qt signals)
